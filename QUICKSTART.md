# 🚀 Advanced College Essay Agent - Quick Start

**Get your advanced AI essay writing tool running in 5 minutes!**

## ⚡ 30-Second Setup

### 1️⃣ Install Python Packages
```bash
pip install -r requirements.txt
```

### 2️⃣ Add Your OpenAI API Key
Edit `.env` file:
```
OPENAI_API_KEY=sk-paste_your_real_key_here
```

Get a free API key: https://platform.openai.com/api-keys

### 3️⃣ Start the Server
```bash
python app.py
```

### 4️⃣ Open in Browser
```
http://127.0.0.1:5000
```

**That's it! You're ready to go! 🎉**

## 🎓 Essay Types Explained

### **Common App (650 words)** - THE BIG ONE
- What you're writing: Your personal story
- What they want: Who you are, how you think, what matters to you
- Our help: Full essay generation, deep analysis, improvement options
- Example: "My grandmother taught me that home is not a place, it's..."

### **Why School (400-500 words)** - SHOW YOU DID YOUR RESEARCH
- What you're writing: Why THIS school specifically
- What they want: Specific knowledge, genuine interest, clear fit
- Our help: Essay generation, names of programs/professors, authenticity check
- Example: "I'm drawn to Princeton's molecular biology labs where..."

### **Coalition (650 words)** - THE ALTERNATIVE
- What you're writing: Your choice of 5 prompts (identity, intellect, communities, etc.)
- What they want: Deep self-reflection and specific examples
- Our help: Full essay generation, structure guidance, authenticity coaching
- Example: "My identity as a first-generation student means..."

### **Supplemental (250-350 words)** - SHORT & SWEET
- What you're writing: A quick answer to their specific question
- What they want: Concise, authentic, insightful
- Our help: Essay generation, word count tracking
- Example: "What activity matters most to you?..."

## 🚨 Common Issues & Fixes

### "ModuleNotFoundError: No module named 'flask'"
```bash
pip install -r requirements.txt
```

### "OPENAI_API_KEY not found"
- Check `.env` file exists
- Check it has `OPENAI_API_KEY=sk-...`
- Check API key is real (not `your_api_key`)
- Restart the server

### "Connection refused" 
- Make sure server is running (`python app.py`)
- Check you're using `http://127.0.0.1:5000` (not localhost)

### "Slow essay generation"
- This is NORMAL - takes 15-30 seconds
- Don't close the browser/kill the server
- It's working, just wait!

### "API key is invalid"
- Go get a NEW key: https://platform.openai.com/api-keys
- Delete old one
- Update `.env` with new key
- Restart server

## 💡 Pro Tips

### 🎯 Get Better Essays
1. **Answer discovery questions fully** - AI is only as good as your input
2. **Be specific** - "I love science" is bad, "I grew up watching my mom's lab notebooks" is gold
3. **Use all improvement options** - try each one, pick best version
4. **Save multiple versions** - you'll change your mind, good to have backup
5. **Read aloud** - you'll catch awkward phrasing and hear if voice is authentic

### ⚡ Save Time
- Use brainstorm for ideas when stuck
- Use why-school template to save time
- Save drafts frequently
- Use improve function rather than rewriting

### 🏆 Write Better Essays
- Specificity > Generic (name the biology program, not "love science")
- Vulnerability > Perfection (share a struggle, not just achievements)
- Reflection > Storytelling (explain what you learned, not just what happened)
- Voice > Formality (sound like you, not a dictionary)
- Growth > Perfection (show how you changed, not that you're already perfect)
